run_sbusr.py
==============

.. program:: run_sbusr.py

在当前终端/命令行窗口中运行

直接运行：

    POSIX（需要权限）::

        ./run_sbusr.py <option>

    Windows（需要权限与文件关联）::

        run_sbusr.py <option>                  

使用 `Python <https://www.python.org>`_ 运行::

    python run_sbusr.py <option>

.. option:: -V, --version

   显示版本信息。

.. option:: -m, --more-detailed-logging

   输出更详细的日志。默认为 ``False`` 。

.. option:: -h, --help

    输出帮助信息
